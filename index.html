Projects and Work




Procedural Spider

A Unity Engine project in which a controllable wall-walking spider uses inverse kinematics (IK) to position its legs to its surroundings in a smart fashion, such that it moves realistically. The user can freely control the spider, which is able to walk on any surface: Walls, corners, ceilings, ... you name it! While moving, the legs dynamically position themselves to the surroundings without the use of any animations, that is all the movement is procedural.
